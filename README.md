# RS Toolkit (core + modules)

This is a starter build setup for a hybrid architecture:
- `dist/rs-core.min.js` (required)
- `dist/modules/rs-modal.min.js` (module example)
- `dist/rs-toolkit.min.js` (all-in-one)

## Install
```bash
npm install
```

## Build
```bash
npm run build
```

## Dev (watch)
```bash
npm run dev
```

## Test
After building, open `test.html` in a browser and check the console.
